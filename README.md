# DTC Veil Brand Workspace

This repository is the shared working folder for the modern bridal veil DTC project.

## Folder Map

- `brand-playbook/` - brand strategy, positioning, product architecture, channel notes, competitor research, and HTML modules.
- `merchandising/` - merchandising logic, SKU planning rules, launch inventory notes, and decision rules.
- `financial-models/` - Excel financial models and launch inventory scenarios.
- `research/` - future market, competitor, SEO, affiliate, creator, and traffic research.

## Current Core Files

- `BRAND_PLAYBOOK_HOME.html` - entry page for the brand playbook.
- `BRAND_PLAYBOOK_FULL.html` - full brand playbook export.
- `financial-models/modern_bridal_veil_financial_model_v8_lean_launch_69_units_startup_fixed.xlsx` - current lean launch financial model.
- `merchandising/bridal_veil_merchandising_summary.md` - plain-language merchandising summary.
- `merchandising/merchandising_decision_rules_v1.md` - merchandising decision rules extracted from source materials.

## Sharing Rule

Keep this repo focused on files the team can safely work from. Do not commit raw textbook PDFs, private downloads, temporary scratch files, or credentials.
