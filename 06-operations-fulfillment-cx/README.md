# Operations / Fulfillment / Customer Experience

## Purpose

This department owns how orders, inventory, shipping, returns, and customer communication actually work.

## Responsibilities

- Inventory receiving
- Storage
- Picking and packing
- Packaging workflow
- Shipping workflow
- Return and exchange handling
- Customer service
- Order issue escalation
- Delivery promise

## Put Files Here

- Receiving checklists
- Packing SOPs
- Packaging specs
- Shipping policies
- Return and exchange policies
- Customer service templates
- Defect handling process
- Inventory count process

## Agent Role

Future agent role: **Operations Agent**.

This agent should translate sales and inventory plans into reliable fulfillment workflows and customer support processes.
