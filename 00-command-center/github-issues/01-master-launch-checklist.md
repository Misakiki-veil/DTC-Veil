# Master Launch Checklist

Use this issue as the shared launch control checklist. Check items directly in GitHub as they are completed.

## Founder Decisions

- [ ] Brand name and relationship to Sovelle
- [ ] Final launch pricing
- [ ] Website path: Shopify-native, content-first, or headless
- [ ] Canonical financial model version

## Cross-Department Gates

- [ ] Gate 1: Domain and brand identity confirmed → SEO clock starts
- [ ] Gate 2: Sample approval by SKU → production go/no-go
- [ ] Gate 3: Demand signal review → final launch buy split
- [ ] Gate 4: Purchase flow QA → launch go/no-go
- [ ] Gate 5: Inventory intake QC → true sellable count is set

## Phase 0: Start Now

- [ ] Confirm brand name and Sovelle relationship
- [ ] Confirm final launch pricing
- [ ] Register domain after brand name is chosen
- [ ] Reserve key social handles
- [ ] Start LLC / EIN / bank / payment setup if needed
- [ ] Start sample order process
- [ ] Create SEO-friendly page structure
- [ ] Create pre-photo content topics
- [ ] Create per-SKU keyword map
- [ ] Create coming-soon / waitlist plan

## Phase 1: Samples, Validation, And Production Decision

- [ ] Receive samples
- [ ] QC samples by SKU
- [ ] Decide pass / revise / delay / remove by SKU
- [ ] Prepare launch shoot plan
- [ ] Prepare production PO after sample approval
- [ ] Review early waitlist/content demand signals
- [ ] Adjust launch buy split if evidence supports it
- [ ] Gate 2 passed: Sample approval by SKU
- [ ] Gate 3 passed: Demand signal review completed

## Phase 2: Build Store During Production

- [ ] Build product pages
- [ ] Add product education and trust content
- [ ] Set up collections
- [ ] Set up email capture
- [ ] Set up payment
- [ ] Set up shipping and return policies
- [ ] Set up analytics
- [ ] QA full purchase flow
- [ ] Gate 4 passed: Purchase flow QA complete

## Phase 3: Pre-Launch

- [ ] Receive first inventory
- [ ] Complete inventory intake QC
- [ ] Record true sellable count by SKU
- [ ] Prepare launch announcement to waitlist
- [ ] Prepare creator seeding
- [ ] Prepare social launch content
- [ ] Prepare small paid test
- [ ] Gate 5 passed: Inventory intake QC complete

## Phase 4: Launch And Early Operations

- [ ] Launch store
- [ ] Send waitlist / early access announcement
- [ ] Start review and wear-photo collection
- [ ] Run first light merch review
- [ ] Track true sellable inventory
- [ ] Apply early reorder floors
- [ ] Review OTB cash before replenishment
- [ ] Start monthly actual vs model review
