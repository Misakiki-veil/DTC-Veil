Source: Claude launch map reviewed on 2026-06-24.

This file is the cross-department launch index. Each department owns its own action list inside its `active-projects/` folder.

## Current Launch Logic

The launch should be managed by gates, not by a vague task list.

Core principle:

Before committing too much inventory cash, use cheaper demand signals such as waitlist, content saves, creator feedback, and per-SKU search intent to improve the launch buy.

## Open Founder Decisions

- Brand name and relationship to Sovelle
- Final launch pricing
- Whether the site starts as Shopify-native or a separate content site/headless path
- Final financial model version to treat as canonical

## Cross-Department Gates

1. Domain and brand identity confirmed → SEO clock starts
2. Sample approval by SKU → production go/no-go
3. Demand signal review → final launch buy split
4. Purchase flow QA → launch go/no-go
5. Inventory intake QC → true sellable count is set
