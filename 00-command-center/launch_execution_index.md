# Launch Master Checklist

Source: Claude launch map reviewed on 2026-06-24.

This file is the cross-department launch checklist. Each department owns its detailed checklist inside its `active-projects/` folder.

## Current Launch Logic

The launch should be managed by gates, not by a vague task list.

Core principle:

Before committing too much inventory cash, use cheaper demand signals such as waitlist, content saves, creator feedback, and per-SKU search intent to improve the launch buy.

## How To Use This Checklist

- `[ ]` means not complete.
- `[x]` means complete.
- A gate should only be checked after the required evidence exists in the relevant department folder.
- When a checkbox changes, commit and push so the team sees the same progress.

## Open Founder Decisions

- [ ] Brand name and relationship to Sovelle
- [ ] Final launch pricing
- [ ] Website path: Shopify-native, content-first, or headless
- [ ] Canonical financial model version

## Cross-Department Gates

- [ ] Gate 1: Domain and brand identity confirmed → SEO clock starts
- [ ] Gate 2: Sample approval by SKU → production go/no-go
- [ ] Gate 3: Demand signal review → final launch buy split
- [ ] Gate 4: Purchase flow QA → launch go/no-go
- [ ] Gate 5: Inventory intake QC → true sellable count is set
