# Google Keyword Planner Insights / Veil SEO 关键词验证

数据来源：用户从 Google Keyword Planner 导出的 `Keyword Stats 2026-06-17 at 19_50_07.csv`

时间范围：June 1, 2025 - May 31, 2026

说明：Google Keyword Planner 的 `Avg. monthly searches` 在很多账号里是区间/桶状数据，不应理解为精确搜索量。这里主要用于判断关键词量级、相对优先级和产品/页面方向。

## 最重要结论

我们之前对 `cathedral veil` 的判断需要上调：它不是普通细分词，而是和 `wedding veil / bridal veil / veil` 同级别的最高量级词。

同时，`mantilla veil` 也进入了最高量级桶，说明它不是小众到只能做内容的词，而是一个有真实搜索需求的高价值细分入口。

## 对首批 SKU 的影响

### 1. Cathedral / Long veil 必须保留，而且权重要提高

之前我们担心 `cathedral veil` 会和 Calily 撞，但 Keyword Planner 显示它是最高量级词。现在的结论不是避开它，而是：

- 产品形态上必须做 cathedral / long veil
- 页面上必须承接 `cathedral veil`
- 用户可读表达可以用 `long wedding veil`
- 差异化要靠 dress pairing / modern bride / anti-template angle

推荐表达：

- `Minimal Cathedral Veil | Soft Long Wedding Veil`
- `Structured Cathedral Veil | Modern Long Bridal Veil`
- `Pearl Cathedral Veil | Long Veil for Satin Dresses`

### 2. Mantilla 需要从“小众 hero 款”上调为“高搜索差异化款”

`mantilla veil` 和 `mantilla` 都显示 50,000 量级，说明 mantilla 不能只当作少量审美款。

但要注意：

- `mantilla` 可能包含宗教/文化头纱搜索，不全是婚礼购买意图
- `mantilla veil`、`mantilla wedding veil`、`mantilla lace veil` 更适合 bridal SEO

策略：

- 保留 Soft Lace Mantilla Veil
- 建独立 `Mantilla Veils` collection
- 加一篇 guide：`What Is a Mantilla Veil? How to Wear It Without Looking Too Traditional`
- 首批库存可从 8 条上调到 10-12 条，具体要看打样成本和 lace 供应链

### 3. Pearl 仍然是安全升级款，但搜索量不如 cathedral/mantilla

`pearl veil` 是 5,000 量级，不是最高量级，但它的优势是：

- 容易被用户理解
- 视觉传播强
- 适合 simple/satin dress
- 适合做 AOV 升级款

结论：Pearl 仍然值得主推，但不要误以为它是最大的 SEO 入口。

### 4. Long veil 是辅助词，不是替代 cathedral 的词

`long veil` 和 `long wedding veil` 都是 5,000 量级，而 `cathedral veil` 是 50,000 量级。

所以：

- 不能只写 `long veil`
- `long wedding veil` 用来降低用户理解门槛
- `cathedral veil` 用来承接高意图 SEO

### 5. Modern / simple 这些差异化词搜索量低，但仍然重要

`modern veil`：500  
`modern wedding veil`：50  
`simple veil`：500  
`simple wedding veil`：500

这说明：

- `modern` 不适合作为第一阶段主 SEO 词
- 但它适合作为品牌语言、页面副标题和内容定位
- SEO 主体仍然要靠 `cathedral veil / mantilla veil / wedding veil / pearl veil / lace veil`

## 关键战略修正

之前我们说：

> Cathedral 是地基，mantilla 是差异化 hero。

现在应修正为：

> Cathedral 是最大地基词之一；mantilla 是同时具备高搜索量和差异化潜力的战略词。

但最终品牌不能只变成：

> Cathedral + Mantilla veil shop

我们的差异化仍然是：

> 用高搜索量的传统 veil 词做入口，用 dress pairing + modern / anti-template bride 的表达完成差异化。

## 具体 SEO 打法

核心原则：用高搜索量词拿入口，用搭配问题和现代审美做差异化。页面必须让 Google 看懂我们卖 `cathedral veil / mantilla veil / wedding veil`，但用户一进来看到的是“这条 veil 适不适合我的婚纱和我的风格”。

## Collection 页面模板

每个 collection 页面都应该包含这些模块，不要只放产品网格：

1. **H1**：直接包含主关键词，例如 Cathedral Veils。
2. **第一屏副标题**：加入差异化，例如 long wedding veils for modern brides who want drama without looking overdone。
3. **产品网格**：至少 3-6 个相关产品，首批不够时可用 waitlist / preorder / coming soon 占位。
4. **How to choose**：解释适合什么婚纱、场地、发型、照片效果。
5. **Style matrix**：按 dress type 推荐：satin、lace、simple、beaded、low back。
6. **FAQ**：回答长度、颜色、固定方式、是否适合户外、是否只戴 ceremony。
7. **Internal links**：链接到 guide 和相关 collection，例如 Cathedral Veils 链到 Pearl Veils、Lace Veils、Veil Length Guide。

## 判断 SEO 是否有效的指标

- **Google Search Console**：先看 impressions 是否增长，不要一开始只看 clicks。
- **关键词覆盖**：cathedral veil、mantilla veil、wedding veil、pearl veil、lace veil 是否开始出现展示。
- **页面表现**：collection 页是否带来 add to cart / quiz starts / swatch kit clicks。
- **内容表现**：guide 页是否把用户导向产品页，而不是只带来阅读。
- **转化辅助**：Veil Finder、Swatch Kit、Stylist Help 是否被 SEO 流量点击。

## 一句话执行原则

不要为了避开 Calily 而放弃 cathedral / mantilla 这种高搜索词；要用同样的高需求入口，讲完全不同的购买理由。
