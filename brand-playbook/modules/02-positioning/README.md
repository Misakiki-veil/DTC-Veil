# 02 品牌定位细分 / Positioning

这个文件夹用来继续细化品牌定位、人群、品牌语气和差异化。

建议子文档：

- `target-customer.md`：目标用户画像
- `brand-positioning.md`：品牌定位语句
- `brand-voice.md`：品牌语气和文案规则
- `customer-segments.md`：用户分层
- `differentiation-map.md`：差异化地图
- `taglines.md`：英文标语和 slogan

核心问题：

- 我们到底服务谁？
- 她为什么选择我们，而不是 Calily / Franc / Etsy？
- 我们的“反传统”应该如何表达得有吸引力但不排他？
