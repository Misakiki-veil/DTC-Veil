# 03 产品架构细分 / Product Architecture

这个文件夹用来继续细化产品线、首发 SKU、价格带、材料和定制服务。

建议子文档：

- `launch-sku-list.md`：首发 SKU 列表
- `launch-pricing-tool.html`：首批 Core SKU 定价决策工具
- `category-map.md`：产品类目地图
- `pricing-ladder.md`：价格带
- `materials-and-trims.md`：材料、蕾丝、珍珠、网纱库
- `bundles.md`：套装和加购组合
- `custom-program.md`：定制服务
- `hero-products.md`：负责品牌感的主推款
- `volume-products.md`：负责走量的基础款

核心问题：

- 首发到底卖什么？
- 哪些款负责流量，哪些款负责利润，哪些款负责品牌记忆点？
- 如何把供应链优势变成产品优势？
