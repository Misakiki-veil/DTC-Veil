建议子文档：

- `tiktok.md`：TikTok 策略
- `instagram.md`：Instagram 策略
- `pinterest.md`：Pinterest 策略
- `seo.md`：SEO 策略
- `google-ads.md`：Google Ads 测试计划
- `affiliate-creators.md`：Affiliate / creator 策略
- `email-sms.md`：Email / SMS flows
- `pr-collabs.md`：PR 和合作
