# 08 销售模型细分 / Sales Model

这个文件夹用来继续细化销量、价格、毛利和增长目标。

建议子文档：

- `traffic-model.md`：流量模型
- `conversion-model.md`：转化率模型
- `aov-model.md`：客单价模型
- `unit-economics.md`：单品成本和毛利
- `monthly-targets.md`：月度目标
- `scenario-planning.md`：保守/基准/强势情景
- `competitor-sales-estimates.md`：竞品销量估算

核心问题：

- 每个月需要多少访问量？
- 需要多少转化率才能盈利？
- 哪些产品负责拉高 AOV？
