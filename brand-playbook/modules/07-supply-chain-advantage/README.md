# 07 供应链优势细分 / Supply Chain Advantage

这个文件夹用来把供应链能力转成产品、营销和转化优势。

建议子文档：

- `supplier-capabilities.md`：供应链能力清单
- `custom-lead-times.md`：定制交付周期
- `quality-control.md`：质量控制
- `material-library.md`：材料库
- `cost-advantage.md`：成本优势
- `rush-orders.md`：加急订单机制
- `limited-drops.md`：小批量上新机制

核心问题：

- 我们比竞品快在哪里？
- 我们比竞品多什么选择？
- 我们如何把成本优势讲成用户价值，而不是低价感？
