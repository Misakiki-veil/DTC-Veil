# 06 独立站漏斗细分 / DTC Funnel

这个文件夹用来继续细化网站结构、转化工具、quiz 和 email funnel。

建议子文档：

- `site-map.md`：网站结构
- `homepage.md`：首页结构
- `product-page.md`：产品页结构
- `collection-pages.md`：集合页结构
- `veil-finder-quiz.md`：Veil Finder Quiz
- `swatch-kit-funnel.md`：Swatch kit 漏斗
- `cart-upsells.md`：购物车加购
- `email-flows.md`：邮件流程
- `trust-stack.md`：review、UGC、材料、售后信任体系

核心问题：

- 用户从哪里进站？
- 她为什么信任我们？
- 她为什么现在下单，而不是继续犹豫？
