# 05 内容体系细分 / Content System

这个文件夹用来继续细化内容栏目、脚本、选题和发布计划。

建议子文档：

- `content-pillars.md`：内容支柱
- `hook-bank.md`：Hook 库
- `script-templates.md`：脚本模板
- `30-day-calendar.md`：30 天内容日历
- `90-day-calendar.md`：90 天内容日历
- `ugc-briefs.md`：UGC creator brief
- `founder-content.md`：创始人内容
- `product-demo-content.md`：产品展示内容

核心问题：

- 我们每天发什么？
- 什么内容负责涨粉，什么内容负责转化？
- 如何把“反传统但不排他”的品牌态度表达出来？
