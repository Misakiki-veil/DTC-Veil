# 婚纱配饰品牌作战手册 / Bridal Accessories Brand Playbook

这是我们后续规划品牌、产品、渠道和增长策略的长期工作台。

核心方向：

- 做一个更适合 Gen Z / young Millennial 新娘审美的现代婚纱配饰品牌。
- 不做泛泛的 wedding brand，而是聚焦 **头纱 + 婚纱造型配饰**。
- 用头纱作为入口产品，再扩展到完整 bridal finishing pieces。
- 组合打法：Calily 的 DTC 增长系统 + Franc 的高级感表达 + 我们自己的供应链速度和成本优势。

## 当前品牌北极星 / Current North Star

英文定位草案：

> Statement veils and bridal finishing pieces for modern brides rewriting the rules.

中文理解：

> 为有自己审美和态度的新娘，提供更有造型感的头纱和婚纱配饰。

更精准的内部目标人群：

> 有审美、有预算意识、社交媒体原生、想要婚礼造型有记忆点，但不想为传统 bridal boutique markup 买单的 Gen Z / young Millennial 新娘。

## 战略支柱 / Strategic Pillars

- 品牌态度：不是 anti-wedding，而是 anti-template。
- 产品方向：经典底子 + 非传统造型表达。
- 价格策略：看起来高级，但比传统 bridal boutique 更可负担。
- 供应链优势：更快定制、更多款式、更好利润空间。
- 内容系统：造型教育、避坑、dress x accessory matching、wedding weekend looks。
- 转化漏斗：swatch kit、quiz、bundle、custom consult、email flows。

## 主要竞品学习 / Competitor Lessons

### Calily Bridal

可学习：

- TikTok/DTC 增长系统。
- “premium veil without bridal markup” 的痛点表达。
- Shopify + Klaviyo + Judge.me + GoAffPro + UGC video + upsell/preorder 的转化工具组合。

可超越：

- 更强审美态度。
- 更完整的婚纱配饰矩阵。
- 更快定制。
- 更系统的 bridal styling education。

### Franc Bridal

可学习：

- 高级 atelier 感。
- 材料和工艺叙事。
- Swatch sample 降低决策门槛。
- Made-to-order 包装成奢华定制体验。
- 统一、优雅的产品命名。

可超越：

- 更快交付。
- 更适合 DTC 的内容引擎。
- 更适合 Gen Z 的造型表达。
- 更有价格梯度和可进入性。
