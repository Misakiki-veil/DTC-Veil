# E-commerce / Website / Conversion

## Purpose

This department owns the shopping experience: how customers understand products, choose the right veil, and complete purchase.

## Responsibilities

- Shopify setup
- Product pages
- Collections
- Navigation
- Quiz or guide logic
- Checkout flow
- Product filters
- Size and length education
- Analytics tracking
- Conversion rate optimization

## Put Files Here

- Shopify setup notes

- Product page wireframes
- Collection structure
- Quiz logic
- Size and length guides
- FAQ and trust signals
- Conversion audits

Future agent role: **E-commerce Agent**.
