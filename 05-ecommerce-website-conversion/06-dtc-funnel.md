# 06 独立站转化漏斗 / DTC Funnel

目的：设计网站体验、转化工具和信任体系。

## 漏斗阶段 / Funnel Stages

### 1. 发现 / Discovery

来源：

- TikTok
- Instagram
- Pinterest
- Google
- Creator links

落地页：

- Product pages
- Collection pages
- Quiz
- Guides
- Swatch kit

### 2. 考虑 / Consideration

工具：

- Veil finder quiz
- Swatch kit
- Dress x veil guide
- Length guide
- Color guide
- UGC videos
- Reviews
- Compare boutique price vs our price

### 3. 转化 / Conversion

工具：

- Bundles
- Rush custom option
- Wedding date field
- Clear lead time
- Installments
- Free shipping threshold
- Cart upsells

### 4. 复购/推荐 / Retention And Referral

工具：

- Review request
- UGC request
- Referral code
- Bride squad discount
- Anniversary content

## 网站必须有的页面 / Must-Have Site Sections

- Shop by veil type
- Shop by wedding mood
- Shop by dress style
- Shop by timeline
- Swatch kit
- Custom veil
- Bridal accessory bundles
- Guides
- Reviews / real brides

## Quiz 结构 / Quiz Structure

问题：

- Wedding date
- Dress silhouette
- Dress neckline
- Dress fabric/detail
- Hair plan
- Venue
- Desired mood
- Budget
- Timeline

输出：

- Recommended veil
- Recommended accessories
- Bundle
- Swatch suggestions
- Email capture
