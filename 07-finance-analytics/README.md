# Finance / Analytics

## Purpose

This department owns the financial model, cash flow, SKU profitability, and performance measurement.

## Responsibilities

- Financial model
- Startup cost tracking
- Cash flow
- Monthly P&L
- SKU profitability
- Inventory cash tied up
- Marketing budget
- Scenario planning
- Performance dashboard

## Put Files Here

- Financial models
- Budget trackers
- Startup cost trackers
- SKU profit trackers
- CAC and payback trackers
- Cash flow forecasts
- Reorder cash capacity analysis
- Monthly performance reports

## Agent Role

Future agent role: **Finance Agent**.

This agent should convert team plans into financial impact, flag cash constraints, and keep assumptions aligned with actual results.
